# 金鹰奖

2024年年10月20日，第32届中国电视金鹰奖在湖南长沙举行，《梦华录》荣获第32届金鹰奖优秀电视剧大奖。另外, 导演`杨阳`（导演）和演员`柳岩`（女配）也分别获得提名。
