# 入围提名


## 白玉兰

2023年05月26日，第二十八届上海电视节白玉兰奖公布入围名单。其中梦华录获`最佳中国电视剧`,`最佳导演`,`最佳编剧(原创)`三项提名。

## 金豪笔编剧之夜
`金豪笔编剧之夜` 为爱奇艺主办，在2023年举办的第三届活动中，《梦华录》剧本获得了`年度热议剧本`和`最佳原创剧本`两个提名。
