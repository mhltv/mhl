# 主创报道

《梦华录》作为一部优秀的群像剧，不仅是主创团队和主演们受到了广泛关注，剧中大大小小的配角也纷纷“出圈”，不管是之前以"龙王"
出名的管云鹏，还是剧中只有几个镜头的”路人“，都以自己专业的演出而被观众喜爱，被外界报道。本文收录《梦华录》整个主创团队不同人员所接受的采访，以及由媒体/自媒体编写的对于部分人员的报道。
